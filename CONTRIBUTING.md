# Contributing to Napstify

Thanks for wanting to help! This is a fun project to recreate the classic Napster experience, so let's keep it simple and enjoyable.

## 🚀 Quick Start

1. **Fork the repo**
2. **Make your changes**
3. **Test it works**
4. **Submit a PR**

That's it! No crazy rules here.

## 💡 What We'd Love

- **Bug fixes** - Something not working? Fix it!
- **UI improvements** - Make it look more like classic Napster/Win98
- **Better fake data** - More realistic usernames, file sizes, etc.
- **Documentation** - Clearer setup instructions
- **Performance** - Make it faster or more efficient

## 🎨 Code Style

- **TypeScript** - Keep using it
- **Tailwind** - For styling
- **Win98 aesthetic** - Maintain the classic look
- **Keep it simple** - Don't over-engineer

## 🐛 Bug Reports

Just open an issue and tell us:
- What you expected to happen
- What actually happened
- How to reproduce it

## 🎵 Feature Requests

Got an idea? Open an issue and describe it! We love creative ideas that fit the Napster vibe.

## 🤝 Pull Requests

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-idea`
3. Make your changes
4. Test it works
5. Submit the PR

No need for formal commit messages or extensive testing - just make sure it works and fits the project!

---

That's it! Have fun and keep the Napster spirit alive! 🎵
